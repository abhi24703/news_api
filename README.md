Setup Instructions 
Prerequisites
- Flutter SDK 3.19.0+
- Dart 3.3.0+
- NewsAPI.org API key
 Installation
1. Clone the repository
```bash
git clone https://github.com/<your-username>/flutter-news-app.git
2. Get api key from https://newsapi.org/
3. Replace your api key in lib\utils\constants.dart
4. run these commands
- flutter pub get
- flutter run
